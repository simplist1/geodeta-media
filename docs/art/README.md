# Geodeta Media app art

These SVG files are temporary PWA icon templates:

- `icon-192.svg` — standard 192×192 launcher icon
- `icon-512.svg` — standard 512×512 launcher icon
- `icon-maskable.svg` — full-bleed Android maskable icon

When final artwork is ready, replace these files or add PNG versions and update `docs/manifest.webmanifest`.

For best Android support, export final PNG files at exactly 192×192 and 512×512 pixels. Keep important artwork inside the central 80% of the maskable icon so Android launchers can crop it safely.
